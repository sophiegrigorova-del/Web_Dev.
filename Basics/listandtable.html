Lists and Tables in HTML

There are two primary types of lists in HTML. The unorderd list (<ul>),which is used for collections when sequenced list does not matter, and the ordered list (<ol>) where the sequence does matter.

Both of these lists use the element list item (<li>) for each item in the list.

Tables

Tables allow you to oraginze data in rows & columns. Semantically, "header" rows should be included in the <head> and the main section of the table should be in a <body> element.

Notice that <h> elements automatically bold the content. These should be used for header cells. The <caption> element in a table gives a caption to the table which is displayed above the table by default. Each row is denoted by the <tr> element, with <td> and <th> elements residing within each row.


One of the harder aspects to master in HTML Tables is the rowspan, which causes the cell to span vertically over rows, and the colspan, which causes the cell to span horizontally over columns.
